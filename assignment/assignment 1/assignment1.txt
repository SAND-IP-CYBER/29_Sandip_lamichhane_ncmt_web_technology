Define the term protocol in context of internet?
>= A protocol is a standarized set of rules and procedures that govern how data is formatted, transmitted and received between devices acting like a common language for computer to ensure seamless communication.
Key aspects: 
Rules and standards: protocol defines syntax (format), semantics(meaning), and timing for data exchange.
Universal language : They allow diverse hardware and software to communicate as long as they follow the same protocol
Examples: Http,TCP,IP.
